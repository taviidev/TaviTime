# Screenshots de TaviTime

## Imágenes necesarias para el README principal

Coloca aquí los archivos con estos nombres exactos:

### 1. `dashboard-empleado.png`
- Vista del dashboard principal de un empleado
- Muestra el widget de fichaje (botón grande)
- Incluye las stats cards (horas hoy, esta semana, etc.)
- **Resolución recomendada**: 1920x1080

### 2. `vacaciones.png`
- Vista del calendario de vacaciones
- Muestra el modal de solicitud de días
- Incluye el balance de días disponibles
- **Resolución recomendada**: 1920x1080

### 3. `reportes.png`
- Vista del informe mensual para inspección de trabajo
- Muestra la tabla de fichajes agrupados
- Incluye el botón de exportar PDF
- **Resolución recomendada**: 1920x1080

### 4. `mobile.png`
- Vista de la PWA en móvil (iOS o Android)
- Muestra la pantalla de fichaje o dashboard
- **Resolución recomendada**: 390x844 (iPhone 13) o similar

## Cómo capturarlas

### En Desktop:
1. Abrir la app en Chrome en pantalla completa (1920x1080)
2. Usar Cmd+Shift+4 (Mac) o herramienta de recortes
3. Guardar como PNG con el nombre indicado

### En Móvil:
1. Abrir la PWA en el móvil
2. Hacer captura con botones del teléfono
3. Enviar al Mac y renombrar

## Tips para buenas screenshots:
- Usar datos de demo (no datos reales de clientes)
- Ocultar información sensible (emails, DNI, etc.)
- Asegurarse de que el diseño se vea limpio y profesional
- Usar tema claro para mejor visibilidad
